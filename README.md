# Automação de emissão de CNVs

Projeto desenvolvido em Python para automatizar a geração e impressão de termos de entrega e recebimento de CNVs.

## Funcionalidades

- Leitura automática de dados via Excel
- Organização e tratamento das informações
- Geração automática de documentos Word
- Salvamento individual dos arquivos
- Impressão automática

## Tecnologias utilizadas

- Python
- pandas
- docxtpl
- openpyxl

## Objetivo

Reduzir o tempo gasto em um processo manual repetitivo realizado diariamente no ambiente de trabalho.

## Resultado

O processo que antes levava mais de 1 hora passou a ser realizado em poucos minutos.

## Como executar

```bash
pip install -r requirements.txt
python main.py
